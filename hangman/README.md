# Hangman Game

## Built With
Python Django

## Installation
1. Install virtual environment
`virtualenv -p /usr/bin/python djangoEnv`

2. Source virtual environment
`source $dir/bin/activate`

3. Install django
`pip install django`

4. Create project
`django-admin startproject hangman`
